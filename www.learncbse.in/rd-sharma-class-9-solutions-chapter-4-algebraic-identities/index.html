If a2 + b2 + c2 – ab – bc – ca = 0, then
(a) a + b = c
(b) b + c = a
(c) c + a = b
(d) a = b = c

⇒  a2 + b2 – 2ab + b2 + c2 – 2bc + c2 + a2 – 2ca = 0
⇒  (a – b)2 + (b – c)2 + (c – a)2 = 0

⇒ a = b

b-c = 0

⇒ c = a
