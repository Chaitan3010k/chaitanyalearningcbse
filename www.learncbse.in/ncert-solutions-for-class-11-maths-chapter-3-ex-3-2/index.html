निम्नलिखित प्रश्नों में से पाँच अन्य त्रिकोणमितीय फलनों का मान ज्ञात कीजिए।

प्रश्न 1.
cos x = \(\frac { -1 }{ 2 }\) , x तीसरे चतुर्थाश में स्थित है।

प्रश्न 2.
sin x = \(\frac { 3 }{ 5 }\) , x दूसरे चतुर्थाश में स्थित है।

प्रश्न 3.
cot x = \(\frac { 3 }{ 4 }\) , x तृतीय चतुर्थाश में स्थित है।
हलः
cot x = \(\frac { 3 }{ 4 }\)
OA = 3 इकाई

प्रश्न 4.
sec x = \(\frac { 13 }{ 5 }\) , x चतुर्थ चतुर्थाश में स्थित है।

प्रश्न 5.
tan x = \(\frac { -5 }{ 12 }\) , x दुसरे चतुर्थाश में स्थित है।

प्रश्न संख्या 6 से 10 तक के मान ज्ञात कीजिए:
प्रश्न 6.
sin 765°

प्रश्न 7.
cosec (-1410)°

प्रश्न 8.
tan \(\frac { 19\pi }{ 3 }\)

प्रश्न 9.
sin (\(\frac { -11\pi }{ 3 }\)).

प्रश्न 10.
cot (\(\frac { -15\pi }{ 4 }\)).
